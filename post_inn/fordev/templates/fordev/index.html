Аутентификация

Каждый запрос к API (за исключение аутентификации) должен содержать аутентификационный токен (access token). Для получения токена необходимо выполнить метод token_create. В ответ вы получите два токена:

  • access token - для запросов, имеет время жизни;
  • refresh token - токен (с большим временем жизни, чем access token), необходим для создания нового access токена, если он больше не действительный. Используйте метод token_refresh_create;
